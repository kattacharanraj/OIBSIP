
## What you need first

1. **Node.js**
2. **MongoDB** — either:
   - Install MongoDB locally and let it run, OR

## How to run

Open **two terminals**.

### 1. Backend

```bash
cd server
npm install
npm run dev
```

On the first run it automatically:
- creates the admin account,
- creates a ready-to-use test user, and
- fills the inventory with starter ingredients.

### 2. Frontend

```bash
cd client
npm install
npm run dev
```

Open the URL it prints (usually `http://localhost:5173`).
